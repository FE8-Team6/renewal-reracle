<h3 align="center">
    REracle 재활용품 별 분리수거 방법 서비스
</h3>
<p align="center">

REracle 사용자들이 쉽고 정확하게 재활용품을 분리수거할 수 있도록 도와주는 웹 기반 서비스입니다. React와 Firebase를 활용하여 개발된 이 서비스는 실시간 데이터 업데이트 기능을 제공합니다.

</p>

## 시작

### 설치 및 실행

```bash
git clone https://github.com/FE8-Team6/renewal-reracle.git
```

```bash
yarn install
```

```bash
yarn dev
```
